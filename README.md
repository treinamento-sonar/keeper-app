# Keeper App

## Sobre o Projeto

O Keeper App é uma aplicação web de anotações desenvolvida com React, que replica as funcionalidades básicas do Google Keep. 

Importante destacar: Este projeto não é de minha autoria conceitual, mas foi desenvolvido como parte dos exercícios práticos do 
The Complete Full-Stack Web Development (https://www.udemy.com/course/the-complete-web-development-bootcamp/) Bootcamp da App Brewery. 

A ideia, estrutura e requisitos do projeto são propostos pelo curso. Meu objetivo ao desenvolver esta aplicação foi aplicar na prática 
os conceitos fundamentais do React. 

## Tecnologias Utilizadas

- Linguagens: JavaScript, CSS e HTML
- Bibliotecas/Frameworks: React, Node.js e Vite
- Build Tools: Vite
- Gerenciador de Pacotes: NPM
- Ferramentas: Git, Github e VS Code 

## Como Executar

1. Pré-requisito: Certifique-se de ter o Node.js instalado. 
2. Clone o repositório: git clone https://github.com/seu-usuario/keeper-app.git
3. Acesse a pasta do projeto: cd keeper-app
4. Instale as dependências: npm install 
5. Execute a aplicação: npm run dev
6. Acesse o navegador: O projeto estará rodando no endereço que aparecer no terminal (normalmente http://localhost:5173).
